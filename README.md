# Sterling

PSE Portfolio Dashboard — Powered by Sterling

## Features

- **Portfolio Overview** — Real-time holdings with P&L tracking
- **Morning Brief** — Sterling's daily market analysis
- **Watchlist** — Buy recommendations with technical/fundamental signals
- **Alerts** — Price drops, news alerts, and dividend reminders
- **News** — Market intelligence with sentiment analysis
- **Dividends** — Calendar view and income projections
- **Discovery** — New stock picks and entry points

## Tech Stack

- Pure HTML/CSS/JS (no frameworks)
- Supabase REST API for data
- GitHub Pages hosting
- Phisix API for live PSEi index
